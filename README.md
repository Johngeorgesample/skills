# Skills

I often hop between my personal and work machines — this repo keeps them in sync. You might find something useful
in here!

## Installation

Install with the [`skills` CLI](https://github.com/vercel-labs/skills), which works with Claude Code, Codex, Cursor, and most other coding agents:

```bash
npx skills add johngeorgesample/skills
```

It will prompt you for which skills to install, which of your agents to install them into, and project vs. global scope. To install everything globally with no prompts:

```bash
npx skills add johngeorgesample/skills --all -g -y
```

To pull updates later:

```bash
npx skills update
```
